# 在 SharePoint 2013 中创建关联

关联是在外部数据实体之间创建关系的一种方式。您可以在两种外部内容类型之间创建关联，从而将外键中的数据连接起来。例如，当您查看客户个人资料时，可能希望查看客户最新下达的订单及每个订单中的商品。
