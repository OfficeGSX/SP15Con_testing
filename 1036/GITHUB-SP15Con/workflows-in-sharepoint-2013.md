# Flux de travail dans SharePoint 2013
Découvrez l'infrastructure de création et les scénarios des flux de travail Workflow Manager Client 1.0 pour SharePoint 2013.
SharePoint 2013 marque l'arrivée de Workflow Manager Client 1.0 comme nouvelle base des flux de travail SharePoint. Fondés sur Windows Workflow Foundation 4, les flux de travail SharePoint offrent une nouvelle gamme de fonctionnalités et d'améliorations. Pour une vue d'ensemble de la nouvelle version des flux de travail SharePoint 2013, voir  [Nouveautés des flux de travail pour SharePoint 2013](what-s-new-in-workflows-for-sharepoint-2013.md).
  
    
    

Pour plus d'informations sur la configuration de Workflow Manager Client 1.0 et son association avec SharePoint 2013 pour un fonctionnement correct, voir  [Installer et configurer le Gestionnaire de workflow SharePoint 2013](set-up-and-configure-sharepoint-2013-workflow-manager.md).
